# bouncing_ball

Simple bouncing ball simulation

# Libraries used
windows.h
graphics.h

# Main features
This is just a simple bouncing ball simulation which i made as a small side project. The ball_pattern.cpp is another variation of program which produces a mesmerising pattern over time which i thought was cool so i created a separate file for it. The main program is pretty simple and barebones as i wanted it to be as lightweight as possible.

The program is opensource so you can do anything you want.
